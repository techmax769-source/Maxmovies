# MaxMovies

A vanilla JS single-page application for streaming and downloading movies.

## Features
- Stream Movies/Series (HLS support)
- Download for Offline Playback (IndexedDB)
- Search & Discovery
- Mock Mode for development

## Setup
1. Clone the repo.
2. Run with a local server (e.g., Live Server in VS Code) because ES6 modules and Service Workers require HTTP(s).
3. Open `index.html`.
